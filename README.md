# Stock-Dashboard
## Install
`pip install -r requirements.txt`

A [Tiingo](https://api.tiingo.com/) free API key is available [here](https://api.tiingo.com/).
## Run
`python app.py`

Go to [http://127.0.0.1:8050/](http://127.0.0.1:8050/) to view the dashboard.
